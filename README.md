# Student Performance Prediction using Machine Learning

## Project Overview

This project predicts student performance using Machine Learning techniques.  
The model analyzes different student-related factors such as study hours, previous scores, sleep hours, extracurricular activities, and practice papers to predict the final performance index.

The project uses the Random Forest Regressor algorithm for prediction and visualization techniques for data analysis.

---

## Dataset Used

Student Performance Dataset

Dataset contains:
- Hours Studied
- Previous Scores
- Extracurricular Activities
- Sleep Hours
- Sample Question Papers Practiced
- Performance Index
